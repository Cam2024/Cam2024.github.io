Recurrent neural networks for aircraft trajectory prediction using ADS-B data
Jinsong Zhang

This project will make use of recurrent neural networks, such as long short term memory (LSTM) and gated recurrent units (GRU) to predict aircraft trajectories using Automatic Dependent Surveillance Broadcast (ADS-B) data, collected by a network of sensors around Europe [1]. This project can be implemented in Matlab or in Python (Tensorflow) [2][3].
